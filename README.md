# abc-company-eda
Employee data analysis project using python
ABC Company Employee Data Analysis:

 Project Overview:
 
This project is about analyzing employee data from ABC Company.
The dataset has 458 rows and 9 columns. It includes details like team, position, age, salary, and height.
 Data Preprocessing:
The height column had some wrong values.
So, we replaced it with random values between 150 and 180 cm.
Then we cleaned the dataset to make sure there are no errors.

 Analysis Tasks:
Employee Distribution by Team;
We checked how many employees are in each team and their percentage.
Employees by Position;
We analyzed how employees are divided based on their job roles.
Age Group Analysis;
We grouped employees into age ranges to find the most common age group.
Salary Expenditure;
We found which team and position spend the most on salaries.
Correlation Between Age and Salary;
We checked if age and salary are related using a graph and correlation value.

Graphical Representations:
Bar chart for team distribution
Bar chart for position distribution
Bar chart for age groups
Bar chart for salary
Scatter plot for age vs salary

Key Insights:
Employees are not equally distributed across teams.
Some positions have more employees, so the company depends on those roles.
Most employees are in the 25–35 age group, showing a young workforce.
Some teams and positions spend more on salaries, so they are important.
Age and salary have a weak relationship, so salary depends more on role and skills.
